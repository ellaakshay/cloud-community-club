# Cloud Community Club – Static Website

## Overview
This project is a simple static website created for a club initiative called **Cloud Community Club (C3)**.
The goal of the website is to introduce the club, share upcoming events, and allow students to join the community.

The website is built using basic web technologies and focuses on a clean and simple user interface.

## Features

* Simple and responsive design
* Navigation bar for easy section access
* About section explaining the club
* Upcoming events section
* Team members section
* Join form with JavaScript interaction
* Deployed online for public access

## Technologies Used

* HTML
* CSS
* JavaScript
